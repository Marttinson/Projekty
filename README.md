# Projekty
Projekty byly vytvořené při samostudiu nebo při studiu na vysoké škole, někde je i zadání od učitelů.

Vetšina projektů obsahuje makefile a lze spustit pomocí příkazu "make run" a pomocí příkazu "make clean" smazat vytvořené soubory. Veškeré testování spuštění proběhlo na linuxu, nezaručuji tedy funkčnost spuštění na windows nebo jiném systému. Každý projekt obsahuje zdrojové a někdy i pomocné soubory.
